Releases and Roadmap

The current release is version 1.3. This release is stable but not mature, and is not feature complete. I do not consider it production quality.

Version 1 is the proof-of-concept version. In it:

  • Basic state machine functionality
  • A common payload dispatcher
  • Completion (including success and error) notification support
  • Adjustable cycle sleep time
  • Support for simultaneous and sequential processing
  • Pluggable processing queues
  • An in-memory queue implementation

Version 2 is the ease-of-use version. In it:

  • Annotation support for declaring state worker methods
  • Cycle callbacks, start and end
  • State processing callbacks, before and after
  • Payload processing timeouts
  • Multi-threading of worker method executions
  • Thread pool support
  • A JPA queue implementation

Version 3 is the expand-the-functionality version. In it:

  • A JPA queue implementation
  • Explore other functionality expansions
